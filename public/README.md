# carwash-saas

Sistema web desarrollado en PHP, JavaScript, HTML y CSS para la gestión integral de un carwash.

## Características
- Gestión de clientes
- Registro de servicios
- Control de pagos
- Reportes básicos

## Tecnologías
- PHP 8+
- MySQL
- JavaScript
- HTML5 / CSS3

## Instalación
1. Clonar el repositorio
2. Configurar el archivo .env
3. Importar la base de datos
4. Ejecutar en servidor local (XAMPP/Laragon)
